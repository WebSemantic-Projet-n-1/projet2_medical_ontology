# Configuration du triplestore

Place ici les fichiers de configuration spécifiques au triplestore choisi, par exemple :

- **Fuseki** : `fuseki.ttl` ou configuration du dataset (nom, persistence).
- **GraphDB** : paramètres du repository (ruleset, etc.).

Ne pas commiter de secrets (mots de passe) ; utiliser des variables d’environnement si besoin.
