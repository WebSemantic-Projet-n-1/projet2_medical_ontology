# Données GO (non versionnées)

Place ici les fichiers OWL téléchargés (deux versions de GO), par exemple :

- `go_2025-10.owl` (ou nom fourni par Zenodo)
- `go_2026-01.owl`

Lien : <https://zenodo.org/records/18422732>

Ces fichiers sont généralement trop volumineux pour le dépôt Git ; le dossier peut être ajouté à `.gitignore` ou les fichiers laissés hors dépôt. Les scripts d’analyse et d’ETL doivent pointer vers ce dossier (chemin relatif ou variable d’environnement).
